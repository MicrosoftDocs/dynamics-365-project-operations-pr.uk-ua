# <a name="whats-new-or-changed-in-project-service-automation-update-release-275-v3"></a>Нові й оновлені можливості в оновленому випуску Project Service Automation 27.5 версії 3

[!include [banner](../includes/psa-now-project-operations.md)]

Ми з радістю повідомляємо про вихід останнього оновлення для програми Project Service Automation для Dynamics 365. Цей випуск містить деякі важливі покращення якості, продуктивності та зручності. Цей випуск сумісний із Dynamics 365 9.x. Щоб інсталювати цей випуск, відкрийте Центр адміністрування Dynamics 365 в Інтернеті й перейдіть на сторінку рішень. Щоб отримати додаткові відомості, див. [Інсталяція, оновлення або вилучення основного рішення](/power-platform/admin/install-remove-preferred-solution).

У цій статті перелічено функції та виправлення, які є новими або зміненими для автоматизації служби project V3, оновлення випуску 27.5. Ця версія має номер збірки V3.10.45.119 і зазвичай надається в складі оновлення за січень 2021 р., яке можна завантажити самостійно.
