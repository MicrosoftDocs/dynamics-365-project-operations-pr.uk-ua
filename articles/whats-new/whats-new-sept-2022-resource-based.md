# <a name="whats-new-september-2022---project-operations-for-resourcenon-stocked-based-scenarios"></a>Що нового у вересні 2022 р. – Project Operations для сценаріїв на основі ресурсів і відсутності запасів

Цю статтю призначено для таких компонентів і програм корпорації Майкрософт Dynamics 365 Project Operations:

## <a name="project-operations-dual-write-maps-updates"></a>Оновлення зіставлень з подвійним записуванням в Project Operations

У наведеній нижче таблиці показано карти подвійного запису, змінені або додані у вересні 2022 року, у вересні 2022 року.

| Зіставлення сутностей | Оновлена версія | Коментарі |
| -------------- | ------------------- | ------------|
| Фактичні дані про інтеграцію Project Operations (msdyn_actuals) | 1.0.0.15 | Ця карта підтримує обробку субпідрядних операцій за допомогою Project Operations для сценаріїв на основі ресурсів. |
| Сутність Project Operations для інтеграції експорту рахунку постачальника проекту (msdyn_projectvendorinvoices) | 1.0.0.2 | Ця карта підтримує обробку субпідрядних операцій за допомогою Project Operations для сценаріїв на основі ресурсів. |
| Сутність Project Operations для інтеграції експорту позиції рахунку постачальника проекту (msdyn_projectvendorinvoicelines) | 1.0.0.5 | Ця карта підтримує обробку субпідрядних операцій за допомогою Project Operations для сценаріїв на основі ресурсів. |

Завжди запускайте найновішу версію карти у своєму середовищі та вмикайте всі пов’язані карти таблиць під час оновлення рішення Project Operations Dataverse та версії фінансового рішення. Деякі функції та можливості можуть працювати неправильно, якщо не активовано останню версію карти. Активну версію зіставлення можна переглянути у стовпці **Версія** сторінки **Подвійне записування**. Щоб активувати нову версію зіставлення, виберіть пункт **Версії зіставлення таблиці**, виберіть найновішу версію та збережіть вибрану версію. Якщо ви настроїли готову таблицю мапи, застосуйте зміни повторно. Щоб отримати додаткові відомості, див. розділ [Керування життєвим циклом програм](/dynamics365/fin-ops-core/dev-itpro/data-entities/dual-write/app-lifecycle-management).

Якщо під час запуску карти виникла проблема, дотримуйтеся вказівок, наведених у [розділі Відсутні стовпці таблиці в розділі Карти посібника з виправлення неполадок](/dynamics365/fin-ops-core/dev-itpro/data-entities/dual-write/dual-write-troubleshooting-finops-upgrades#missing-table-columns-issue-on-maps) подвійного запису.
