# <a name="vendor-invoice-lines-for-time"></a>Позиції в рахунку постачальника для часу

_**Застосовується до:** розгортання Lite: від угоди до рахунків-проформ_

Рахунок-фактура постачальника в корпорації Майкрософт Dynamics 365 Project Operations може містити рядки рахунків-фактур постачальників на час. Керівники проектів можуть використовувати рядки рахунків-фактур постачальників протягом часу, щоб зафіксувати витрати на час субпідрядника на проекти.

Рядки рахунків-фактур постачальників на час можуть посилатися або не посилатися на лінію субпідряду на час. Якщо рядок рахунка-фактури постачальника за часом посилається на субпідрядник, керівники проектів зможуть зіставити та перевірити час, який виставляється лінією рахунка-фактури постачальника проти часу, який фіксується субпідрядниками та затверджується керівниками проекту на проекті.

У таблиці нижче наведено відомості про поля в рядках рахунків-фактур постачальників за час.

| Поле | Опис | Функціональний вплив |
| --- | --- | --- |
| Ім'я | Назва рядка накладної постачальника, щоб допомогти з ідентифікацією. | Це ім’я відображатиметься як перший стовпець у всіх підстановках, які базуються на рядках рахунків-фактур постачальників. |
| Опис | Короткий опис послуг, які виставляються постачальником у рядку рахунків-фактур постачальників. | Не надано |
| Субпідрядний сервісний договір | Субпідряд, за яким спочатку були замовлені послуги. | Коли для рахунку-фактури постачальника вибрано субпідрядник, усі рядки в рахунку-фактурі постачальника успадковують цей вибір. Рахунок-фактура постачальника не може містити рядки рахунків-фактур постачальників, які посилаються на різні субпідряди. |
| Лінія субпідряду | Лінія субпідряду, за якою були замовлені послуги. Перелік ліній субпідряду, які можна обрати, обмежується лініями на обраному субпідряді. | Якщо в рядку рахунка-фактури постачальника вибрано рядок субпідряду за час, значення за промовчанням для **полів «Проект»,** **«Роль»** і **«Книжковий ресурс**» вводяться з відповідних полів у рядку субпідряду. Якщо вибраний рядок субпідряду містить значення в **полях Проект**, **Роль** і **Книгорозповсюджувач**, значення відповідних полів у рядку рахунка-фактури постачальника не можуть відрізнятися від цих значень. |
| Дата транзакції | Дата, коли фактична вартість рядка накладної постачальника буде зафіксована на проекті. | Не надано |
| Клас транзакції | Значення за замовчуванням **Час**. | Значення **Time** вказує на те, що рядок рахунка-фактури постачальника використовується для запису суми рахунка-фактури за часом субпідрядника. |
| Project | Назва проекту, на якому використовувалися послуги, яким виставляється рахунок-фактура. | Це поле є обов’язковим для заповнення, і його не можна залишати порожнім. |
| Завдання | Ім’я завдання проекту, для якого використовувалися служби, для яких використовується рахунок-фактура. Це поле доступне, лише якщо вибрано проект. Вибір завдання проекту не є обов’язковим. | Якщо це поле залишити порожнім, керівник проекту може зіставити рядок рахунка-фактури постачальника з часом, записаним ресурсами субпідрядника для будь-якого завдання проекту. Якщо рядок рахунка-фактури постачальника не посилається на рядок субпідряду, а це поле залишається пустим, фактична вартість, створена рядком рахунка-фактури постачальника, не буде пов’язана з будь-якими необлікованими фактичними даними про продаж. У цьому випадку, якщо настроєно виставлення рахунків на основі завдань, можливо, витрати не вдасться виставляти кінцевому клієнту. |
| Роль | Роль ресурсів субпідряду, час яких виставляється на рахунок. | У цьому полі вказується роль, яку виконують ресурси субпідряду, час яких виставлено в рахунку-фактурі постачальника. |
| Планований ресурс | Ім’я субпідрядника, час якого виставляється. Вибір книжкового ресурсу не є обов’язковим. | Якщо це поле залишити пустим, керівник проекту може зіставити рядок рахунка-фактури постачальника з часом, записаним будь-яким ресурсом, що належить постачальнику, у рядку рахунка-фактури постачальника. |
| Кількість | У рядку рахунка-фактури введіть кількість годин виставлення рахунків постачальником. |Не надано |
| Група одиниць вимірювання | Значення за замовчуванням – **група** одиниць часу, яку не можна змінити. | Не надано |
| Одиниця вимірювання | Значення за промовчанням – це базова одиниця годин від групи одиниць часу. Це значення можна змінити, щоб придбати його в будь-якій одиниці групи одиниць часу, наприклад день або тиждень. | Комбінація значень **ролі** та **одиниці** використовуватиметься як значення за замовчуванням або обчислене значення для поля Ціна **за одиницю** в рядку рахунка-фактури постачальника. |
| Ціна за одиницю | У ціні за одиницю товару за промовчанням використовується комбінація **значень ролі** та **одиниці** з прейскуранту проекту, яка застосовується до дати транзакції рядка рахунка-фактури постачальника. | Якщо ціна для відповідного прейскуранту проекту встановлюється в одиницю, яка відрізняється від одиниці в рядку рахунка-фактури постачальника, система використовує перетворення одиниці для розрахунку ціни за одиницю товару. |
| Проміжний результат | Це поле, доступне лише для читання, обчислюється як *ціна* за одиницю товару &times;*·*, якщо значення введено як у поле «Кількість», **і** в поле «Ціна за **одиницю товару**». Якщо одне або обидва ці поля пусті, можна ввести значення в це поле. | Не надано |
| Податок із продажу | Введіть суму податку з продажу. | Не надано |
| Загальна сума | Загальна сума рядка рахунка-фактури постачальника, включаючи податки. Це поле обчислюється як *податок із продажів за підсумками* + *·*. | Не надано |
