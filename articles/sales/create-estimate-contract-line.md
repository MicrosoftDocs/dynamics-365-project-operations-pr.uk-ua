# <a name="estimate-a-project-contract-line"></a>Оцінка сервісної роботи за проектним договором

_**Застосовується до:** Project Operations для сценаріїв на основі ресурсів і відсутності запасів_ 

У системі Dynamics 365 Project Operations сервісна робота за договором на основі проекту має докладні відомості, які дозволяють оцінити вартість і потенційний дохід від виконання сервісної роботи за договором.

Щоб оцінити сервісну роботу за договором на основі проекту, перейдіть на вкладку **Відомості про виконання сервісної роботи за договором** у розділі **Сервісна роботі за договором** на основі проекту.  Є два способи визначення оцінки сервісної роботи за договором на основі проекту.

   - Створіть оцінку безпосередньо в сервісній роботі за договором способом додавання відомостей про сервісну роботу за договором.
   - Створіть проект і план проекту, потім пов’яжіть цей проект і завдання із сервісною роботою за договором проекту. Це запускає процедуру, завдяки якій ви зможете імпортувати кошторис плану проекту до сервісної роботи за договором на основі компонентів, включених до сервісної роботи за договором.

## <a name="create-an-estimate-directly-on-a-project-contract-line"></a>Створіть кошторис безпосередньо в сервісній роботі за договором за проектом

Щоб створити кошторис безпосередньо в сервісній роботі за договором за проектом. виконайте наведені далі кроки.

1. Перейдіть до сервісної роботи за договором і виберіть вкладку **Відомості про виконання сервісної роботи за договором**. Створені вами на цій вкладці рядки відображаються як **Договірна вартість** для цієї **Сервісної роботи за договором**. 
2. На вкладеній сітці **Відомості про сервісну роботу за договором** виберіть пункт **Нові відомості про сервісну роботу за договором**. Відкриється повзунок «Швидке створення». На сторінці **Відомості про сервісну роботу за договором** доступні наведені далі поля.

| Поле | Розташування | Опис | Вплив на наступні етапи |
| --- | --- | --- | --- |
| **Опис** | **Швидке створення** | Опис для певної оцінки. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично. |
| **Клас транзакції** | **Швидке створення** | Список класів транзакцій наведено на вкладці **Загальні** у позиції на основі проектів у сервісному договорі. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично. |
| **Вибрати продукт** | **Швидке створення** | Застосовується, коли класом транзакції є **Матеріали**. За бажанням ви можете зазначити цей рядок кошторису, як рядок для **Наявного** (каталожного) продукту або **Доданого** продукту. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично. |
| **Продукт** | **Швидке створення** | Ідентифікатор продукту з каталогу продуктів. Це поле вмикається, лише якщо ви вибрали **Наявний продукт** у полі **Вибрати продукт**. Ідентифікатор використовується для витягнення ціни збуту із проектного прайсу за сервісним договором. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично. |
| **Доданий продукт** | **Швидке створення** | Текстове поле для введення найменування продукту. Це поле вмикається, лише якщо ви вибрали **Доданий** у полі **Вибрати продукт**.| Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично. |
| **Роль** | **Швидке створення** | Роль особи, яка виконує цю роботу або здійснює ці витрати. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично.|
| **Категорія** | **Швидке створення** | Категорія роботи або витрат. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично.|
| **Дата початку** | **Швидке створення** | Дата початку роботи. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично. |
| **Дата завершення** | **Швидке створення** | Дата закінчення роботи. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично. |
| **Компанія, яка надає ресурс** | **Швидке створення** | Компанія, яка надає ресурс, або юридична особа, яка несе ці витрати та надає ресурс для роботи над ним. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично, при цьому воно також використовується при витягненні вартості. |
| **Одиниця ресурсів** | **Швидке створення** | Організаційна одиниця, яка надає ресурс, яка несе ці витрати та надає ресурс для роботи над ним. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично, при цьому воно також використовується при витягненні вартості. |
| **Розклад одиниць** | **Швидке створення** | Група одиниць вимірювання роботи, продукту або витрат. Одиниці вимірювання належать до розкладу одиниць або групи одиниць вимірювання. Наприклад, *милі* та *кілометри (км)* є одиницями вимірювання, що належать до групи одиниць, які описують відстань. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично. |
| **Одиниця вимірювання** | **Швидке створення** | Одиниця роботи, продукту або витрат. | Для вартості використовується значення за промовчанням пов’язаних відомостей сервісної роботи за договором, яке створюється автоматично. |

| **Ціна за одиницю** | **Швидке створення** | Ставка для виставлення рахунку роллю, яка виконує роботу, ціна за одиницю продукції або ціна збуту продукції, або категорії витрат. Значення за промовчанням для **Часу** ґрунтується на поєднанні значень виміру ціноутворення в ціновій позиції для ролі проектного прайса, який є чинним станом на дату початку. Для поля **Витрати** значення за промовчанням береться з налаштування ціни для категорії транзакцій у прайсі проекту, який є чинним станом на дату початку. Якщо метод ціноутворення для категорії транзакцій не є **ціна за одиницю**, значення за замовчуванням відсутнє, й це поле залишається порожнім. Для продуктів значення за промовчанням цього поля ґрунтується на рядку **Позиції прайса** в прайсі проекту, який є чинним станом на дату початку.| Ставка витрат ролі, яка виконує роботу, або вартість одиниці в категорії витрат, або вартість одиниці продукту. Значення за промовчаннням для **Часу** ґрунтується на поєднанні значень виміру ціноутворення для рядка ціни ролі прайса вартості організаційної одиниці, який є чинним станом на дату початку. Для **Витрат** значення за промовчанням для цього поля ґрунтується на рядку ціни категорії прайсу вартості підрозділу організації, який є чинним станом на дату початку. Якщо метод ціноутворення для категорії транзакцій не ціна за одиницю, значення за замовчуванням відсутнє, і це поле лишається пустим. Для продуктів значення цього поля за промовчанням ґрунтується на рядку **Позиція прайса** прайсу витрат підрозділу організації, який є чинним станом на дату початку.|
| **Прогнозований податок** | **Швидке створення** | Користувач вводить орієнтовну суму податку на цю роботу або витрати. | &nbsp; |
| **Сума** | **Швидке створення** | Значення в цьому полі можна додати, якщо поля **Кількість** і **Ціна** залишаються порожніми. Якщо поля **Кількість** і **Ціна** заповнені, поле **Сума** є лише для читання й розраховується як **(Кількість \* Ціна за одиницю) + Податок**. | &nbsp; |

## <a name="update-prices-on-contract-line-details"></a>Оновіть ціни у відомостях сервісної роботи за договором

Якщо змінити ціни в проектному прайсі, який додається до сервісного договору, або в прайсі витрат підрозділу, який відповідає за залучення клієнтів, ви можете оновити ціни в окремих відомостях сервісної роботи за договором із урахуванням відповідних змін. На сторінці **Сервісний договір** виберіть **Перерахувати**. Відображається попередження, яке повідомляє вам, що ціни для всіх сервісних робіт за цим договором скинуто. Виберіть **Так**, щоб оновити ціни як у відомостях збуту, так і у відомостях вартості сервісних робіт за договором.

## <a name="access-contract-line-details-for-cost"></a>Отримайте доступ до відомостей про вартість сервісних робіт за договором.

На вкладці **Відомості про сервісну роботу за договором** виберіть рядок у сітці, щоб на панелі інструментів вкладеної сітки відобразилися справи. Першою справою на панелі інструментів вкладеної сітки є **Відкрити відомості про витрати**. Щоб побачити пов’язані норми витрат і суму для цих відомостей сервісної роботи за договором, виберіть пункт **Відкрити відомості про витрати**. 

> [!NOTE]
> Зміна компанії із ресурсного забезпечення, підрозділу ресурсного забезпечення, значень кількості, дат, ролі або категорії у відомостях сервісної роботи за договором у розділі **Вартість** також змінює відповідні значення у відомостях сервісної роботи за договором у розділі **Збут**.

## <a name="currency-on-contract-line-details-for-cost-and-sales"></a>Валюта сервісної роботи за договором для вартості та збуту

Відомості про сервісну роботу за договором для **Збуту** задають валюту за замовчуванням із проектного прайсу, який є чинним станом на дату відомостей сервісної роботи за договором.

Відомості про сервісну роботу за договором для **Вартості** задають валюту за замовчуванням із прайсу підрозділу, який відповідає за залучення клієнтів за сервісним договором, який є чинним станом на дату відомостей сервісної роботи за договором для **Вартості**.

При розрахунку рентабельності виконується конвертація сум у відомостях сервісної роботи за договором для **Вартості** та **Збуту** в базову валюту середовища для складання звітів про загальні фактичні та потенційні норми дохідності за сервісним договором.

> [!NOTE]
> Через відсутність даних про обмінні курси станом на дату набуття чинності можуть відбуватися помилки, пов’язані з округленням грошових сум, а також зміною норм дохідності. Використовуйте ці розрахунки лише за проектними сервісними договорами, оскільки вони є приблизними значеннями й не є фактичними даними, які можна використовувати для нормативної або іншої звітності, для якої потрібна висока точність округлення та інформація про дати чинності обмінних курсів валют.
